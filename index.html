What is STUDA?

STUDA is a cross-platform application built with Flutter that streamlines the entire examination workflow — from registration to report cards.

Offline-First Architecture

All student data is stored locally using SQLite. No internet connection required for daily operations. Works even in remote areas.

Secure Login System

App password protection with 30-day session expiry. SMTP password works as a recovery option. Your data stays safe.

Instant Report Generation

Generate class summaries and individual report cards with one click. Export as PDF or print directly. Includes grade distribution and statistics.

Automated Backups

Configure SMTP email once and get automatic daily backups. Manual one-click backup also available. Restore from any backup file.

Everything You Need

Manage students, marks, grades, and reports in one unified system.

Registration

Add students by name, class, and stream in seconds.

Marks Entry

Enter subject marks. Grades, totals, and positions calculated automatically.

Auto Ranking

Per-subject and overall student rankings computed instantly.

Report Cards

Individual A4 report slips with marks, grades, ranks, and signature section.

Class Summary

Full exam report with rankings, grade summary, and class statistics.

Custom Setup

Define classes, streams, subjects. Each class can have its own exam title and year.

How to Use STUDA

Get up and running in minutes with these simple steps.

1
Set Up Your School

Go to Setup → School Profile and enter your school name, address, exam title, and year. Configure your App Password and SMTP email for automatic backups.

2
Add Classes, Streams & Subjects

Under Setup → Classes / Streams / Subjects, define your academic structure. Each class can have its own exam title and year, subject list, and stream divisions.

3
Register Students

Go to Register and add students by name, class, and stream.

4
Enter Marks

Go to Marks, select a class and student, then enter marks for each subject. Grades, totals, and positions are calculated automatically.

5
Generate Reports

Go to Report for a Class Summary or individual Report Cards. Print or save as PDF.

6
Backup Your Data

Set up SMTP email for automatic daily backups, or use the Backup Now button on the Dashboard. Restore from a previous backup anytime.
